Tiến độ dự án PC-Builder-Project (30/11/2025)

1. Đã hoàn thiện các chức năng quản lý sản phẩm cho admin:
   - Thêm, sửa, xóa (soft-archive) sản phẩm
   - Hiển thị danh sách sản phẩm, lọc, tìm kiếm, phân trang
   - Chọn sản phẩm hiển thị lên trang bán hàng

2. Đã hoàn thiện chức năng quản lý đơn hàng cho admin:
   - Hiển thị danh sách đơn hàng, lọc, tìm kiếm, phân trang
   - Xem chi tiết đơn hàng, cập nhật trạng thái, nhập lý do hủy, xóa đơn hàng đã hủy

3. Đã thiết kế UI quản lý user cho admin:
   - Trang /admin/users hiển thị danh sách user (bảng: tên, email, vai trò, ngày tạo, thao tác)
   - Tìm kiếm, lọc theo vai trò
   - Đã thêm mục Users vào nav và sidebar admin

4. Đã tạo API trả về danh sách user cho admin (GET /api/admin/users)
   - Đã sửa lỗi schema, hiển thị đúng dữ liệu user

5. Các bước tiếp theo (chưa thực hiện):
   - Thiết kế UI chi tiết user
   - Thiết kế chức năng chỉnh sửa/xóa user
   - Tích hợp phân quyền, trạng thái, tìm kiếm nâng cao cho user

==> Dự án đang tạm dừng ở bước: Đã hoàn thiện UI + API danh sách user cho admin, chuẩn bị chuyển sang chi tiết/chỉnh sửa/xóa user.

Khi tiếp tục, chỉ cần đọc lại file này để nắm tiến độ và thực hiện các bước tiếp theo.
