# Lưu ý về CSS ảnh hưởng nút (button)

- File: `src/app/css/async-gallery.css`
- Rule `.asyncGallery button` sẽ làm TẤT CẢ button bên trong `.asyncGallery` bị mất màu nền, border, text (font-size: 0).
- Nếu overlay `.asyncGallery` tồn tại trong DOM (dù ẩn), rule này vẫn có thể ảnh hưởng đến các button con nếu selector không đủ chặt hoặc overlay không bị unmount đúng cách.
- Khi tạo nút mới, LUÔN kiểm tra file này hoặc dùng class riêng/style inline `!important` cho nút quan trọng.
- Nếu có thể, hãy sửa selector thành `.asyncGallery > .gallery__Button` để chỉ ảnh hưởng nút gallery.
- Nếu gặp lỗi nút bị trắng/mất text, kiểm tra lại rule này đầu tiên!

**Ghi chú này dành cho tất cả thành viên phát triển dự án.**
